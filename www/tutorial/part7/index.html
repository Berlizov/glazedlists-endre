Part 7 - JComboBox

Because Glazed Lists is widget-independent, it is straightforward to set up a list or selection widget to display them. We have already created formatted tables for our list, now we will display it in a JComboBox.

Rendering a JComboBox

For JComboBox display we can either use the default behaviour which displays an object's toString() value, or we can create a custom renderer, similar to what we did for the table. Glazed Lists comes with a sample renderer, IconComboBoxRenderer which allows you to use an icon for each element. We're going to use the default renderer so we need to fix the toString() value of our ProgrammingLanguage object. 

import com.odellengineeringltd.glazedlists.*;

class ProgrammingLanguage implements Filterable, Comparable {
    
    ...
    
    public String toString() {
        return name;
    }
}

The JComboBox List

We now need to replace the JTable in our program with a JComboBox. We will keep the filters, although you may need to decide whether that makes sense for your application. 

    public void display() {
        ListComboBoxModel listCombo = new ListComboBoxModel(customFilteredLanguages);
        
        JFrame frame = new JFrame("Programming Languages");
        frame.getContentPane().setLayout(new GridBagLayout());
        frame.getContentPane().add(new JLabel("Filter"), ...);
        frame.getContentPane().add(filteredLanguages.getFilterEdit(), ...);
        frame.getContentPane().add(customFilteredLanguages.getObjectOrientedCheckBox(), ...);
        frame.getContentPane().add(customFilteredLanguages.getVirtualMachineCheckBox(), ...);
        frame.getContentPane().add(new JComboBox(listCombo), ...);
        frame.setSize(640, 480);
        frame.show();
    }

Just as Glazed Lists can be displayed in a JTable and JComboBox, it is also possible to create other widgets that provide a view of a EventList. For example, it is possible to get Glazed Lists working with Eclipse/AWT widgets. And if you develop bindings for Glazed Lists to another widget, please submit it so that other developers can use it!
