Demo launcher application which allows running all other demo applications.

To include a new demo:

  • The demo class must have public access and provide a normal main(String[]) method
  • The main method must not call System.exit() or use JFrame.EXIT_ON_CLOSE for the default close operation if the system property in_launcher is set to a non-null value.
    Example:
    public static void main( String[] args ) {
	// run test...

	// when finished...
	if ( System.getProperty( "in_launcher" ) == null ) System.exit( 0 );
}
    or 
    public static void main( String[] args ) {
	JFrame my_test_frame = ...

	if (System.getProperty("in_launcher") == null) {
		my_test_frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
	}
	else {
		my_test_frame.setDefaultCloseOperation(JFrame.DISPOSE_ON_CLOSE);
	}

	// run test...
}
  • Add the following to demos.properties:
    1. A property (for example: "aaa") to demos.properties with the display name of the class.
    2. (Optional) A property with the name from the previous step followed by ".desc" (for example: "aaa.desc") containing a description of the demo.
    3. A property with the name from step 1 followed by ".class" (for example: "aaa.class") containing the name of the class containing the main method for the demo.
