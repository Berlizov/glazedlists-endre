Demo launcher application which allows running all other demo applications.

To include a new demo:

  • The demo class must have public access and provide a normal main(String[]) method
  • The main method must not call System.exit() or use JFrame.EXIT_ON_CLOSE for the default close operation if Launcher.runningInLauncher() returns true.
    Example:
    public static void main( String[] args ) {
	// run test...

	// when finished...
	if ( !Launcher.runningInLauncher() ) System.exit( 0 );
}
    or 
    public static void main( String[] args ) {
	JFrame my_test_frame = ...

	if (!Launcher.runningInLauncher()) {
		my_test_frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
	}
	else {
		my_test_frame.setDefaultCloseOperation(JFrame.DISPOSE_ON_CLOSE);
	}

	// run test...
}
  • Add the following to launcher.properties (in the resources/demo directory):
    1. A property (for example: "aaa") with the display name of the class.
    2. A property with the name from step 1 followed by ".class" (for example: "aaa.class") containing the name of the class containing the main method for the demo.
    3. (Optional) A property with the name from the previous step followed by ".desc" (for example: "aaa.desc") containing a description of the demo.
    4. (Optional) If the demo requires SWT, add a property with the name from the pervious step followed by ".requires_swt" (for example: "aaa.requires_swt"). No value is necessary, any value for the property will indicate the SWT is required.
